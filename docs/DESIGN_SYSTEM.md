# PalpNet — Design System

## Paleta de Cores

| Token | Cor | Hex |
|---|---|---|
| `primary` | Verde principal | `#5ACB3D` |
| `primary-dark` | Verde escuro | `#2E8F2D` |
| `white` | Branco | `#FFFFFF` |
| `background` | Fundo escuro | `#0F172A` |

## Tipografia

| Papel | Fonte |
|---|---|
| Principal | **Manrope** (Google Fonts) |
| Pesos usados | 400 (Regular), 600 (SemiBold), 700 (Bold), 800 (ExtraBold) |

## Identidade Visual

- **Favicon:** Balão de conversa + bola de futebol
- **Conceito:** Comunicação social + esporte
- **Tom:** Moderno, vibrante, noturno (dark mode prioritário)

## Componentes Planejados (Fase 3)

- Botões: primário (verde), secundário (outline), ghost
- Cards: grupo, competição, palpite, patrocinador
- Badges: resultado (acerto, placar exato, errou), conquistas
- Avatar com ranking e nível
- Tabela de ranking estilizada
- Modais de palpite
- Notificações toast
- Banners de patrocinadores

## Diretrizes

- Dark mode como padrão (fundo `#0F172A`)
- Gradientes sutis em verde para elementos de destaque
- Animações leves via Tailwind (transition, animate-pulse)
- Responsivo: mobile-first
